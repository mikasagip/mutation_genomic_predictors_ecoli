In this folder the original data file from wei et al 2022 is the 41467_2022_32353_MOESM5_ESM excel file. 

Within this file there are separate sheets for the different growth protocols. 

The selected sheets were exported individually, these are the rest of the excel files in this folder. 

The merge_wei.py file takes in these individual files and creates a merged file called all_wei.csv which is automatically moved to the foster_wei folder.
