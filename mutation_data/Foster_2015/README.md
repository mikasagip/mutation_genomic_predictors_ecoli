Within this folder, the original data file from foster et al, 2015 is the BPS_foster_et_al_2015. 

This contains sepreate sheets within it for the different lines. 

Selected pages were then exported individually, these are all named *_BPS where * is the line name. 

merge_foster.py then merges these individual sheets and produces a file called data_foster.csv which is automatically moved to the foster_wei folder.
