# mutation_genomic_predictors_ecoli
This project creates a multivariate statistical model to identify independent genomic predictors of mutation density in E. coli.

* properties

* stats_and_graphs
  * kendalls_tau_matrices
  * kendalls_heatmaps
  * any order for the rest
