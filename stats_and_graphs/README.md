The kendalls_tau_matrices R file takes in the all_properties.csv as input as outputs all the model csv files.

The kendalls_heatmaps R files takes in all the model csv files and outputs heatmap graphs into the heatmap_outputs folder. 

The bar_graphs, bars_composite, bars_pairs_composite, asymmetry_figure, gerdes_essnetiality_figure, middle_graph, and pec_essentiality_figure R files take in the all_properties.csv file and output figures into the figure_outputs folder.
